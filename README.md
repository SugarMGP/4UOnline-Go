# 4UOnline-Go
校学生会权益服务后端系统

---
## 项目简介

本项目主要使用以下技术构建：
* 编程语言：Go
* Web 框架：Gin
* 文件存储：MinIO
* 日志管理：Zap
* 数据库：MySQL
* 缓存：Redis
* 认证与授权：Session

点此查看[项目具体说明](docs/README.md)
